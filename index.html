PS1 Character Demo

C Python Blender Playstation 1 Tools Programming Graphics Programming
A rigged and animated character tech demo for the Playstation 1, I've always wanted to give back to the console of my childhood and this is one step closer. After creating the character from start to finish in Blender, I wrote an exporter in Python to output the model in a custom format optimised for the Playstation 1's fixed graphics pipeline. Ex: the vertices are sorted by the bones they are attached to, allowing the renderer to update the vertices' positions with just one pass over the vertex data.

Global Illumination Renderer with Temporal Sampling

C++ OpenGL OpenCL Graphics Programming
Real-time renderer made as part of my thesis. Implements a global illumination algorithm (IGI) to be run on the GPU. Previous frames are reused via temporal sampling to further optimise the renderer for dynamic environments. This is done by splitting up light bounce sources into seperate sets across a number of frames, meaning that only a fraction of light bounce sources are calculated per frame, and the rest of the light are re-projected from previous framebuffers to the new camera orientation
